UX/UI Case Study

★ Featured
Web App · Front-End Development · Interaction Design

Ojo AI — Generative Chat Interface

The Problem: Standard conversational AI interfaces often look sterile and clinical. Users experience "blank canvas paralysis" when faced with a generic empty chat box, reducing engagement and slowing onboarding.

The Solution: Designing and developing the Ojo AI Chat Tool from scratch using a "Neural Dark" aesthetic to reduce eye strain, accented with Light Maroon to naturally guide the user's focus. I implemented "Progressive Disclosure" via Suggested Prompt Pills to instantly educate users on the AI's capabilities.

My Role: Conceptualized the UI architecture → Built the high-fidelity structure in Vanilla HTML/CSS using CSS Grid and Flexbox → Programmed advanced JavaScript to simulate authentic generative token-streaming without relying on external libraries.

Outcome: A highly responsive, glassmorphic chat prototype that demonstrates a deep understanding of modern interactive UI design, CSS variables, and raw DOM manipulation.
